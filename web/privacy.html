Personvernerklæring

Sist oppdatert: 1. mars 2026

TourFlow («vi», «oss», «vår») er utviklet av Complete Drums / Michael Thøgersen. Denne personvernerklæringen beskriver hvordan vi samler inn, bruker og beskytter dine personopplysninger når du bruker TourFlow-appen.

1. Hvilke data vi samler inn

  • Kontoinformasjon: E-postadresse og navn ved registrering.
  • Bookingdata: Informasjon du legger inn om gigs, tilbud, kunder, kontakter, turneer og utøvere.
  • Meldinger: Innhold i direktemeldinger og gruppechat innad i appen.
  • Teknisk data: Enhetstype og operativsystem for feilsøking.

2. Hvordan vi bruker dataene

  • Levere og forbedre tjenesten.
  • Autentisere brukere og sikre tilgang.
  • Sende varsler relatert til dine bookinger.

3. Lagring og sikkerhet

Alle data lagres sikkert hos Supabase (EU-region) med kryptering i transit og i hvile. Tilgang er begrenset med Row Level Security (RLS) slik at brukere kun ser data de har rettigheter til.

4. Deling med tredjeparter

Vi selger eller deler ikke dine personopplysninger med tredjeparter. Data deles kun med tjenesteleverandører som er nødvendige for drift:

  • Supabase — database og autentisering
  • Google Routes API — ruteberegning (kun adresser, ingen persondata)

5. Dine rettigheter

I henhold til GDPR har du rett til å:

  • Be om innsyn i dine data
  • Kreve retting eller sletting av data
  • Trekke tilbake samtykke
  • Eksportere dine data

6. Dataoppbevaring

Vi oppbevarer data så lenge kontoen din er aktiv. Ved sletting av konto fjernes alle tilknyttede data innen 30 dager.
